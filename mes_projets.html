L'opinion publique sur les nouvelles technologies : vers une remise en question du progrès continu ?

Analyse de données descriptives

Objectif du Projet

L’objectif est d’expliquer pourquoi certaines personnes rejettent les nouvelles technologies malgré leur diffusion croissante. Le travail cherche à tester si ce rejet s’accentue avec le temps, s’il est lié à une méfiance envers les nouvelles technologies, et si la capacité d’adaptation des individus (exposition au changement) influence leur rapport à l’innovation.

Méthodologie

L’étude combine deux sources de données : le World Value Survey de 1981 à 2022 et un questionnaire original administré à 247 répondants. Deux types d’analyses statistiques ont été menés : une régression linéaire simple pour mesurer l’évolution des opinions dans le temps, puis une régression linéaire multivariée pour tester les hypothèses liées aux inquiétudes et à la capacité d’adaptation.

Technologies Utilisées

création d’un score de réticence construit à partir de réponses normalisées modèles de régression (simple et multivariée) Python

DataInsight

Plateforme d'analyse et visualisation de données

Objectif du Projet

Créer une plateforme permettant aux entreprises d'importer, analyser et visualiser leurs données métier sans compétences techniques avancées.

Méthodologie

Approche centrée utilisateur avec tests utilisateurs réguliers. Développement en Python avec intégration de bibliothèques de data science.

  • Collecte et prétraitement des données avec Pandas
  • Analyse statistique avec Scikit-learn et Statsmodels
  • Visualisations interactives avec Plotly et D3.js
  • Interface utilisateur avec Streamlit
  • Déploiement avec Docker sur AWS

Technologies Utilisées

Python Pandas Scikit-learn Streamlit D3.js AWS Docker

EcoTrack

Application mobile de suivi d'empreinte écologique

Objectif du Projet

Développer une application mobile permettant aux utilisateurs de suivre et réduire leur empreinte carbone grâce à des défis écologiques et des conseils personnalisés.

Méthodologie

Développement cross-platform avec React Native. Intégration d'APIs externes pour le calcul d'empreinte carbone.

  • Conception UX/UI avec Adobe XD
  • Développement frontend avec React Native
  • Backend en Firebase pour l'authentification et le stockage
  • Intégration d'API de données environnementales
  • Tests sur appareils iOS et Android

Technologies Utilisées

React Native Firebase Redux API REST Jest Expo
